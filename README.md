#spinning

>Spinning animation for CLI apps

<img width="173" src="spinner.gif">

##Install

```
$ pip install spinning
```

##Usage

```py
import spinning
spinning.spin()		# Spins for 5 seconds by default
spinning.spin(3)		# Spins for 3 seconds
```
